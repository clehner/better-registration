better-registration
===================

Chrome extension for fetching classes and registering. 

To install:
-----------

Put everything in a folder. Then, 

`Chrome → Menu → Tools → Extensions → ✓ Developer Mode → Load Unpacked Extension → Select the folder and you're done.
`
